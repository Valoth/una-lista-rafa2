# Projeto Lógica de Programação em Java

Este projeto reúne exercícios desenvolvidos em Java com foco na prática de lógica de programação, utilizando entrada de dados, estruturas condicionais, repetição e manipulação de strings.

---

## Objetivo

* Praticar conceitos básicos de programação em Java
* Trabalhar com entrada de dados utilizando `Scanner`
* Aplicar estruturas condicionais e de repetição
* Utilizar funções matemáticas com a classe `Math`
* Manipular strings e caracteres

---

## Projetos

### Impressao.java

Imprime uma saída formatada no console utilizando caracteres especiais (ASCII art).

---

### DistanciaEntrePontos.java

Calcula a distância entre duas coordenadas geográficas na superfície da Terra.

* Conversão de graus para radianos
* Uso de funções trigonométricas (`Math.sin`, `Math.cos`, `Math.acos`)

---

### ContadorCaracteres.java

Conta diferentes tipos de caracteres em uma string informada pelo usuário.

* Letras
* Números
* Espaços
* Outros caracteres

---

### MenuTentativas.java

Simula uma questão de múltipla escolha com controle de tentativas.

* Uso de `switch` e `do-while`
* Respostas com alternativas de A a D
* Limite de 3 tentativas
* Feedback de acerto ou erro
